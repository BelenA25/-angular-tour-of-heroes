RECURSOS PARA DESARROLLADORES

ELEMENTOS HTML

Elemento Descripcion Documentacion
<html> Representa la raiz del documento. html

Elementos genericos

Elemento Descripcion Documentacion
<div> Crea secciones o agrupa contenido. div
<span> Sirve para aplicar estilo al texto. span

Elementos semanticos

Elemento Descripcion Documentacion
<header> Describe el contenido de la cabecera. header
<article> Contenido independiente de la pagina web. article
<section> Contenido de un tema en especifico. section
<nav> Elementos de navegacion. nav
<aside> Contenido relacionado de manera indirecta al contenido que lo rodea. aside
<footer> Pie de pagina para el elemento body o el ancestro de la categoria SECTIONING. footer
<main> Contenido dominante del documento. main

Manejo de texto

Elemento Descripcion Documentacion
<h1>,<h2>,<h3>,...,<h6> Niveles de encabezado. h1 representa un encabezado mas grande y h6 mas pequeño. encabezados
<p> Parrafos de texto. p
<br> Salto de linea, avanza a la siguiente linea de la pagina. br
<hr> Transicion entre parrafos de un tema relacionado. hr
<wbr> Oportunidad de romper la palabra para continuar en la siguiente linea. wbr
<sup> Superindice, permite crear y definir exponentes, letras superiores y numeros ordinales. sup
<sub> Subindice, permite crear numeros al pie, formulas matematicas y quimicas. sub
<strong> Contenido de mucha importancia. Normalmente se muestra en negrita y el lector de pantalla cambia la tonalidad de voz. strong
<em> Denota enfasis en una palabra. El navegador lo muestra en cursiva. em
<mark> Resalta texto de interes. mark
<q> Cita corta en linea, que usualmente es mostrada con el uso de comillas. q
<blockquote> Cita en bloque cuyo contenido es una cita extensa. blockquote
<cite> Titulo de un trabajo que esta citado en la pagina. cite
<dfn> Termino a definir del parafo. dfn
<abbr> Proporciona la descripcion de una abreviatura mediante el atributo tittle. abbr
<small> Define letra mas pequeña. small
<ins> Indica partes de un texto que ha sido añadidos. ins
<del> Texto eliminado. del
<s> Tachado de texto. s
<code> Indica el contenido en formato de codigo de computadora. code
<samp> Salida de computadora. samp
<kbd> Fragmento de texto que indica entrada de usuario desde teclado. kbd
<pre> Presenta el texto tal y como esta escrito en el codigo html. pre
<address> Informacion de contacto. address
